# 新郑

用[郑码](https://baike.baidu.com/item/%E9%83%91%E7%A0%81/589192)的字根和拆字方法做新的输入法。

## 拆字表

### 完整表
xingzheng.tsv

### 简化表
xingzheng-abyz.tsv

完整表的简化版

一字最多四码，若超过四码，则首尾各取两码。

特例：
- 器：口口大口
- 囂：口口頁口
- 嘂：口口凵口
- 嚚：口口臣口
- 鼵：臼巛穴大
- 鼳：臼巛目大

### 特殊码
用于消除重码
- 重：重码键
- 能：功能键
- 空：什么也不按
