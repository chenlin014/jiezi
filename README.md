# 新郑

用[郑码](https://baike.baidu.com/item/%E9%83%91%E7%A0%81/589192)的字根和拆字方法制作新的输入法。

## 拆字表

### 完整表
xingzheng.txt

### 简化表
xingzheng-abyz.txt

最多取四码，若超过四码，则首尾各取两码。

特例：
- 器：口口大口
- 囂：口口頁口
- 嘂：口口凵口
- 嚚：口口臣口
- 鼵：臼巛穴大
- 鼳：臼巛目大

### 特殊码
用于消除重码
- 重：重码
- 能：功能键
- 左：左右结构
- 下：上下结构
- 空：什么也不按
- 正：正體字
- 简：简体字
- 和：日本漢字，新字体
- 喃：越南漢字，喃字
- 韓：朝鮮漢字
