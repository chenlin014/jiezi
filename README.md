# 新郑

源于[郑码](https://baike.baidu.com/item/%E9%83%91%E7%A0%81/589192)的新输入法。

## 码表
放置处：table

### 解字码
jiezi.tsv

### 输入码
shuru.tsv

### 代码
shuru-abyz.tsv

简化自输入码。

特例：
- 囂：口口頁口
- 嘂：口口凵口
- 嚚：口口臣口
- 凹：圍凵
- 凸：圍冂

### 特殊码
- 重：重码
- 能：功能键
- 简：简码
- 空：一只手不按

## 按式表
放置处：chordmap

早林式：cl.tsv
